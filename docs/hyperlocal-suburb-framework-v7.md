# Hyperlocal Suburb Framework V7

## Goal

Improve suburb authority with useful local context that helps customers prepare access, parking, route timing, and property details before requesting a quote.

## Pages Affected

- `/removalists-glenelg/`
- `/removalists-salisbury/`
- `/removalists-adelaide-cbd/`
- `/removalists-marion/`
- `/removalists-norwood/`

## Exact Implementation Rules

- Use broad public context only.
- Include council or local-government context.
- Include access and parking considerations.
- Include road and logistics considerations.
- Include common property types.
- Link naturally to service, price, guide, and quote pages.

## Framework Fields

- Council/context note.
- Access and parking considerations.
- Local road/logistics considerations.
- Common property types.
- Services available in that area.
- Related service, price, guide, and quote links.

## Unsafe Claims To Avoid

- No exact parking statistics unless verified.
- No traffic data unless sourced.
- No fake office or depot claims.
- No doorway-style suburb pages.
- No repeated exact-match anchor stuffing.

## Follow-Up Actions

- Add new suburbs only when there is a real operational difference.
- Keep Glenelg coastal, Salisbury north, CBD tower, Marion south, and Norwood east pages as the quality bar.

## Owner Confirmation Required

- None for broad public local context.
- Address-specific access or permit claims require confirmation.
