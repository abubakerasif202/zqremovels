# ZQ Removals Premium Site Template Kit

This kit generates a reusable premium front-end and SEO structure for `zqremovals.au` without changing the current production pages.

## Visual Thesis

Clean white surfaces, deep navy framing, and amber conversion moments position ZQ Removals as a premium logistics brand instead of a commodity removalist.

## Content Plan

1. Hero: strong brand presence, fixed-price promise, clear conversion.
2. Proof: trust cues, service coverage, and operational confidence.
3. Detail: service depth, route logic, suburb specificity, and FAQs.
4. Final CTA: fixed-price quote and direct phone conversion.

## Interaction Thesis

1. Sticky header keeps the phone number and quote CTA visible.
2. Desktop dropdowns and mobile disclosures support hub-and-spoke crawling without hiding important links.
3. Scroll-snap testimonials create slider behavior without JavaScript dependencies.

## Files

- `site-architecture.md`: flat URL map and internal linking model.
- `zq-premium.css`: shared CSS variables, layout system, and component styling.
- `homepage-template.html`: homepage shell with proof, services, testimonials, and conversion sections.
- `service-hub-template.html`: hub page for service clusters such as interstate removals.
- `route-spoke-template.html`: local or route spoke page with logistics detail and schema-friendly FAQs.
- `media/*.svg`: lightweight image placeholders with descriptive usage targets.
