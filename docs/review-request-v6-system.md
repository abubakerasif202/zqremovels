# Review Request V6 System

Brand: ZQ Removals

## Rules

- No incentives for reviews
- No review gating
- Ask every real customer fairly
- No incentivized reviews
- Never write reviews for customers
- Never pressure customers
- Do not fabricate reviews
- Never ask for only positive reviews
- Never ask customers to edit a review because of a complaint

## SMS template

`Hi [Name], thanks for choosing ZQ Removals for your move. If you were happy with the service, could you leave an honest Google review here: [link]. It helps local Adelaide customers find a reliable removalist.`

## Email template

Subject: Thanks from ZQ Removals

Body:

`Hi [Name], thanks again for booking ZQ Removals. If you have a moment, please leave an honest review here: [link]. We read every review and use feedback to improve our service.`

## WhatsApp template
