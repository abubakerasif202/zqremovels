Yes. Add the route, timing, and service type in the message so we can review the job correctly.

What if I do not know every detail yet?

Send what you know now, especially the suburbs, date window, and rough inventory. We can clarify the rest after the first review.

Should I call instead of using the form?

Call if the move is urgent or the access is unusual. Use the form when you want the full brief captured in one message first.
