Implementation Notes

The shortest safe merge was to route the concepts, not overwrite the live homepage.

That keeps the current revenue pages stable while still making the premium direction fully reviewable inside the production build pipeline.

Build-safe integration

The concept pages are declared in site-src/pages.json and ship through the same generator as the rest of the site.

Published assets

The screenshot references and shared concept stylesheet now resolve from public site paths under /media/stitch/.

Clean follow-up path

If one of these screens should become a live commercial page later, it can be promoted route-by-route instead of replacing the homepage in one step.
